Automation Project

Jenkins-based OLT Configuration Backup Automation

A production-like automation system running in a lab environment, designed for long-term, scheduled OLT configuration backup across multiple vendors.

Project Overview

This project addresses the operational need to reliably back up configuration data from multiple Optical Line Terminals (OLTs) in a laboratory environment.

Instead of running ad-hoc scripts manually, the system is built on top of a Jenkins server to provide scheduling, execution history, and failure visibility.

Architecture & Workflow

  • Jenkins server deployed inside lab network
  • Each OLT mapped to an independent Jenkins job
  • Scheduled execution via cron triggers
  • Python scripts connect to OLT via SSH
  • Configuration output stored and versioned
