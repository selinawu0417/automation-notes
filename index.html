A Jenkins server deployed in the lab environment to automate scheduled OLT configuration backup tasks across multiple vendors.

  • Automated SSH login and configuration dump
  • Multi-vendor OLT support (Nokia / Huawei / ZTE)
  • Per-device job isolation and execution history
  • Stable long-term operation with success / failure tracking
