Automation Tools

PON / WAN / TR-369 Automation Tools

A collection of practical automation tools developed for real-world network testing, packet analysis, and protocol validation.

以下工具皆為實務開發，原始碼維護於 Private GitHub Repository， 可於面試或需求時提供存取。

OMCI Parser & Analysis Tool

Parses OMCI console logs or PCAP files, decodes Managed Entities according to ITU-T G.988, and exports Wireshark-aligned Excel reports.

OMCI log / PCAP 解析工具，對齊 G.988 與 Wireshark 解碼結果。

OMCI G.988 Excel Private Repo

OMCI VLAN Tagging Decoder

Decodes VLAN Tagging Operation Table with bit-level interpretation aligned to Wireshark and ITU-T G.988 definitions.

VLAN Tagging 位元層解析，完整呈現 Inner / Outer Filter & Treatment。

VLAN OMCI Binary Decode Private Repo

OLT Configuration Backup Tool

Automatically connects to OLT via SSH and collects running configuration for backup and audit purposes.

支援 Nokia / Huawei OLT 設定自動化備份。

OLT SSH Automation Private Repo

WAN IPv6 Packet Checker

Validates WAN IPv6 behavior including SLAAC, DHCPv6-PD, Router Advertisement lifecycle, and recovery scenarios.

IPv6 WAN 行為驗證與封包檢查。

IPv6 WAN PyShark Private Repo

AI Packet Diff Tool

Compares two packets or captures and highlights field-level differences, behavioral changes, and potential root causes.

封包差異比對，結合規則與 AI 輔助說明。

Packet Diff AI OMCI / WAN Private Repo

TR-369 USP Agent Test Tool

Verifies USP Agent and Controller connectivity, message flow, and basic management operations.

USP Agent / Controller 測試工具，支援 MQTT。

TR-369 USP MQTT Private Repo

TestRail Automation Uploader

Automatically uploads structured test cases to TestRail with proper field mapping and project configuration.

測試案例自動上傳與 TestRail API 整合。

TestRail API Automation Private Repo

OMCI Log → PCAP Converter

Converts OMCI console logs into PCAP format for Wireshark analysis and further automation.

Console log 轉 PCAP，方便封包分析。

OMCI PCAP Wireshark Private Repo
